Throughout the First Cloud Journey internship program, I have documented my progress and the knowledge accumulated each week. Below is a summary of the tasks performed over the 12 weeks:

#### **[Week 1 (04/17/2026 – 04/23/2026)](1.1-week1/)**
- Create an AWS account and learn basic services.
- Practice with IAM, VPC, and Amazon EC2.
- Learn about cost management and AWS support plans.

#### **[Week 2 (04/24/2026 – 04/30/2026)](1.2-week2/)**
- Learn about Amazon VPC and networking models on AWS.
- Practice VPN Site-to-Site, Subnet, Route Table, Internet Gateway, and NAT Gateway.
- Configure Security Groups and Network ACLs.

#### **[Week 3 (05/01/2026 – 05/07/2026)](1.3-week3/)**
- Learn about Amazon EC2 and key components: Instance Types, AMI, Backup, Key Pair, and EBS.
- Practice launching EC2 instances, configuring storage, and managing secure connections.

#### **[Week 4 (05/08/2026 – 05/14/2026)](1.4-week4/)**
- Connect VPC with VS Code, test connection, and learn basic EC2.
- Learn to transform a blank EC2 instance into a fully functional application server and use VPC Reachability Analyzer.
- Configure EC2 Instance Connect Endpoint and AWS Systems Manager Session Manager for secure access without direct SSH.
- Set up CloudWatch Monitoring & Alarms to monitor system health and raise alerts.

#### **[Week 5 (05/15/2026 – 05/21/2026)](1.5-week5/)**
- Learn and practice converting applications from Monolithic to Microservices and Cloud-Native on AWS.
- Build Serverless applications using AWS Lambda, API Gateway, DynamoDB, and S3, and manage workflow via SQS, SNS, and Step Functions.
- Develop RESTful and GraphQL APIs, integrate SPA frontend, and configure authentication using Amazon Cognito.
- Build deployment automation using AWS SAM, AWS CodePipeline, and CI/CD; use CloudWatch and X-Ray for monitoring.
- Explore AI services on AWS to assist data processing, automate functions, and enhance user experience.

#### **[Week 6 (05/22/2026 – 05/28/2026)](1.6-week6/)**
- Package applications using Docker; build and manage Docker Images for AWS deployment.
- Deploy containerized apps on Amazon Lightsail Containers and Amazon EKS; configure runtime, resources, and check connectivity.
- Migrate Monolithic app to Microservices using Docker and AWS Fargate; manage service discovery and verify scalability.
- Build CI/CD pipelines with AWS CodePipeline and GitHub to automate building, testing, and deployment.
- Configure monitoring, resource management, and security policies for container environments.

#### **[Week 7 (05/29/2026 – 06/04/2026)](1.7-week7/)**
- Learn about Amazon EC2 and the role of virtual servers in deploying applications on AWS.
- Practice launching Amazon Linux EC2 instances, selecting AMI, Instance Types, and configuring Key Pairs.
- Configure EC2 networking including VPC, Subnets, Public IPs, and Security Groups for connectivity and security.
- Manage EC2 instances via AWS Console, monitor system status, and check network parameters.
- Learn remote access via SSH and essential components for deploying apps on Amazon EC2.

#### **[Week 8 (06/05/2026 – 06/11/2026)](1.8-week8/)**
- Research Amazon Bedrock and learn how RAG (Retrieval-Augmented Generation) models work.
- Read, analyze, and translate technical documents related to AWS and AI technologies.
- Analyze business requirements, formulate ideas, and propose the overall project architecture.
- Integrate Amazon Textract with Amazon S3 to extract data from files stored on the cloud.
- Develop AWS Lambda functions to automate data processing combining Textract and Amazon Bedrock.
- Integrate Amazon DynamoDB and perform integration testing on the complete data pipeline.

#### **[Week 9 (06/12/2026 – 06/18/2026)](1.9-week9/)**
- Translate and study technical blog posts about AWS Carbon Reporting, BYOL, and Sustainability.
- Complete the Project Proposal for the AI Invoice Scanner project.
- Test and finalize the functionality of the AI Invoice Scanner project.
- Design the detailed system architecture diagram using Draw.io.
- Write technical documentation and progress notes on Notion.
- Screenshot IAM, S3, and Lambda configurations for project reporting.

#### **[Week 10 (06/19/2026 – 06/25/2026)](1.10-week10/)**
- Learn Data Governance principles: access management, data security, and data quality on the cloud.
- Practice using Amazon QuickSight to connect data sources, build dashboards, and generate analytical reports.

#### **[Week 11 (06/26/2026 – 07/02/2026)](1.11-week11/)**
- Finalize the invoice processing pipeline using Amazon Textract, Amazon Bedrock, and DynamoDB.
- Optimize AWS Lambda functions to enhance data processing performance.
- Complete Amazon QuickSight dashboards for visualization and statistics.
- Review AWS configurations (IAM, S3, Lambda) and reinforce security policies.
- Perform Integration Testing and troubleshoot errors during system operation.
- Prepare technical documentation, presentation slides, and live demo product for final review.

#### **[Week 12 (07/03/2026 – 07/09/2026)](1.12-week12/)**
- Compile final report, presentation slides, and format according to the FCJ template.
